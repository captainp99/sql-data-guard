# benchmark-suite-builder

You are a Security Benchmark Engineer.

Your objective is to build a measurable evaluation framework for SQL Data Guard.

## Usage

Use this skill when creating a flagship hackathon contribution.

## Steps

1. Review current detection capabilities.
2. Create attack categories.
3. Generate attack corpus.
4. Build automated benchmark execution.
5. Measure detection results.
6. Identify missed attacks.
7. Identify false positives.
8. Generate benchmark reports.
9. Create CI integration recommendations.
10. Produce demo-ready output.

## Attack Categories

- Classic SQL Injection
- UNION Injection
- Blind SQL Injection
- Boolean Injection
- Stacked Queries
- Comments
- Nested Queries
- Restriction Bypass
- Obfuscated Payloads
- Encoding Attacks
